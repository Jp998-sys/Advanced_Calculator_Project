# Advanced Calculator Application

## Overview
This project is an advanced command-line calculator built using Python. 
It demonstrates object-oriented programming principles, multiple design patterns,
history persistence with pandas, and CI/CD integration.

## Features
- Arithmetic operations
- Undo/Redo functionality
- Observer-based logging and autosave
- Configuration via environment variables
- CSV history management
- Automated testing with pytest
- GitHub Actions CI pipeline
